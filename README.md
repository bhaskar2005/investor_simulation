# Investor Simulation

Simulate how top investors (a16z, Peter Thiel, Sam Altman, YC, Naval Ravikant, Balaji Srinivasan) would react to your startup idea using Gemini AI.

## Features

- Enter your startup pitch and get feedback from 6 famous investor archetypes
- See interest level, valuation range, investment amount, and detailed feedback
- Minimal, modern UI (Next.js, React, Tailwind)
- Gemini 3 Flash API integration (with fallback)

## Getting Started

### 1. Clone the repo

```bash
git clone <your-repo-url>
cd investor_simulation
```

### 2. Install dependencies

```bash
npm install
```

### 3. Set up your Gemini API key

- Get your API key from: https://aistudio.google.com/api-keys
- Create a `.env.local` file in the project root:

```
GEMINI_API_KEY=your_api_key_here
```

### 4. Run the development server

```bash
npm run dev
```

- Open [http://localhost:3000](http://localhost:3000) in your browser
